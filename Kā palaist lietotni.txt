vienu mazu .txt failu
vienu mazu .pdf
vienu mazu .png vai .jpg

Ieraksti, piemēram:
Šis ir mans pārbaudes fails
Saglabā kā parbaude.txt


2)atvert projekta mapi terminali - 
cd C:\Users\User\OneDrive\Documents\GitHub\projekta-darbs-2026-seglins-klieviens

python -m pip install -r requirements.txt

4) Palaid lietotni.  Vajadzētu atvērties GUI logam / aplikācijai.
